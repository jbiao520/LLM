## Why

为 `topics/embedding/` 下的 Word Embedding 和 Position Embedding 添加 Python 代码示例。当前 `examples/` 目录为空，缺少实际可参考的代码。添加示例代码可以帮助学习者将理论与实践结合，理解 LLM 中真实使用的 embedding 技术。

## What Changes

- 在 `topics/embedding/word-embedding/examples/` 添加词嵌入示例代码
- 在 `topics/embedding/position-embedding/examples/` 添加位置嵌入示例代码
- 示例涵盖 LLM 中常用的 embedding 算法调用
- 代码包含完整的中英文注释

## Capabilities

### New Capabilities

- `embedding-code-examples`: Embedding 模块的 Python 代码示例，包含 Word Embedding（Word2Vec、GloVe、BERT embedding）和 Position Embedding（Sinusoidal、RoPE）的实际调用示例

### Modified Capabilities

（无现有能力需要修改）

## Impact

- 新增 `topics/embedding/word-embedding/examples/` 下的 Python 文件
- 新增 `topics/embedding/position-embedding/examples/` 下的 Python 文件
