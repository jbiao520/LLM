## Why

需要一个结构化的 Python 项目来系统学习大语言模型（LLM）相关的各种知识点。目前项目目录为空，缺乏基础的项目结构和学习组织框架。现在初始化项目可以为一后续的学习和知识管理打下良好基础。

## What Changes

- 初始化 Python 项目基础结构（pyproject.toml、虚拟环境配置）
- 创建知识领域分类的目录结构
- 配置代码质量工具（ruff、mypy）
- 添加项目说明文档

## Capabilities

### New Capabilities

- `project-structure`: 定义项目的目录结构和文件组织方式，支持多知识领域的分类管理
- `python-config`: Python 项目配置，包括依赖管理、虚拟环境、代码质量工具配置

### Modified Capabilities

（无现有能力需要修改）

## Impact

- 新增项目根目录下的配置文件（pyproject.toml、.python-version 等）
- 新增多个知识领域目录
- 新增 README.md 项目说明文档
