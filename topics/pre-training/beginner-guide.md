# 预训练（Pre-training）科普版

> 面向零基础读者的预训练入门指南

## 一句话概括

**预训练就像"海量阅读"，让模型通过阅读大量文本学会理解和生成语言。**

## 从一个问题开始

ChatGPT 为什么能回答各种问题、写代码、翻译语言？

难道有人把所有知识都"编程"���去了？**不是的！**

模型是通过**阅读海量文本**自己"学会"的：
- 读过维基百科 → 知道历史、地理、科学
- 读过代码 → 学会编程
- 读过书籍 → 理解故事和逻辑

**这就是预训练：让模型通过大规模阅读学会语言。**

## 预训练的目标

### 核心任务：预测下一个词

模型在预训练时做的事情很简单：**猜下一个词**。

```
输入: "今天天气真"
目标: 猜出下一个词是 "好"

输入: "1 + 1 = "
目标: 猜出下一个词是 "2"

输入: "Python 是一门"
目标: 猜出下一个词是 "编程语言"
```

### 为什么这有效？

要准确预测下一个词，模型必须：
1. **理解语法**："他正在__" → 需要填动词
2. **理解语义**："苹果是__" → 需要理解"苹果"的含义
3. **记住知识**："中国的首都是__" → 需要知道是北京
4. **推理能力**："如果下雨，那么__" → 需要逻辑推理

**预测下一个词是一个"伪装"的综合学习任务！**

## 预训练数据

### 数据来源

| 来源 | 内容 | 作用 |
|------|------|------|
| Common Crawl | 网页数据 | 通用知识、多样表达 |
| 维基百科 | 结构化知识 | 事实、定义 |
| 书籍 | 长文本 | 逻辑、故事 |
| 代码 (GitHub) | 编程 | 编程能力 |
| 论文 (arXiv) | 学术 | 专业知识 |
| Reddit/论坛 | 对话 | 口语表达 |

### 数据规模

| 模型 | 训练数据量 |
|------|-----------|
| GPT-2 | 40 GB 文本 |
| GPT-3 | 570 GB 文本 |
| LLaMA | 1.4 TB 文本 |
| GPT-4 | 估计 10+ TB |

### 数据处理

原始数据不能直接用，需要：

1. **过滤**：去除低质量、有害内容
2. **去重**：避免重复内容
3. **分词**：将文本转换为 token
4. **混合**：平衡不同来源的比例

## 计算资源

### 预训练需要多少算力？

| 模型 | 参数量 | 训练算力 | GPU 时长 |
|------|--------|---------|---------|
| GPT-2 | 1.5B | ~256 V100 天 | |
| GPT-3 | 175B | ~3,500 V100 年 | |
| LLaMA-7B | 7B | ~82,000 A100 小时 | ~1 个月 |
| LLaMA-70B | 70B | ~1,700,000 A100 小时 | ~2 年 |

### 成本估算

训练一个 7B 模型：
- 租用 8 × A100 (80GB)
- 时间：~1 个月
- 成本：$50,000 - $100,000

**这就是为什么大公司才能训练大模型！**

## 缩放定律（Scaling Laws）

### 核心发现

模型性能与三个因素相关：

1. **模型大小**（参数量）
2. **数据大小**（token 数量）
3. **计算量**（训练步数）

### 基本规律

```
性能提升 ∝ 模型大小^0.5
性能提升 ∝ 数据量^0.5
```

**结论**：想要更好的模型，要么增大模型，要么增加数据！

### Chinchilla 定律

最优训练配置：
```
数据量 ≈ 20 × 模型参数量
```

例如：
- 7B 模型 → 需要约 140B tokens
- 70B 模型 → 需要约 1.4T tokens

## 预训练 vs 微调

| 对比 | 预训练 | 微调 |
|------|--------|------|
| 数据量 | 海量 (TB级) | 较少 (GB级) |
| 目标 | 学会语言 | 学会特定任务 |
| 时间 | 周到月 | 小时到天 |
| 成本 | 极高 | 较低 |
| 输出 | 基础模型 | 专用模型 |

## 预训练模型的"涌现"能力

随着规模增大，模型会出现"涌现"能力：

| 模型大小 | 能力 |
|---------|------|
| 小 (<1B) | 基本语言模式 |
| 中 (1-10B) | 翻译、摘要 |
| 大 (10-100B) | 推理、编程、数学 |
| 超大 (>100B) | 复杂推理、少样本学习 |

**这些能力不是显式训练的，而是"涌现"出来的！**

## 总结

| 概念 | 通俗理解 |
|------|----------|
| 预训练 | 海量阅读学会语言 |
| 下一个词预测 | 猜下一个词的游戏 |
| 训练数据 | 模型的"课本" |
| 缩放定律 | 越大越好 |
