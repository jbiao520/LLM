# 视觉语言模型（Vision-Language Model）科普版

> 面向零基础读者的视觉语言模型入门指南

## 一句话概括

**视觉语言模型让 AI 既能"看图"又能"说话"，实现图像理解和对话。**

## 从一个问题开始

你有一张照片，想问 AI：

- "这张图里有什么���"
- "那个人穿什么颜色的衣服？"
- "图中有几只猫？"

**传统图像分类：** 只能说是"猫"还是"狗"

**视觉语言模型：** 能像人一样描述图片、回答问题！

## 核心组件

### 1. 视觉编码器

提取图像特征（详见 [../vision/](../vision/)）

```
图像 (224×224×3)
     │
     ▼
视觉编码器 (CLIP/ViT)
     │
     ▼
图像特征 (196 × 768)
```

### 2. 适配器/投影层

把视觉特征对齐到语言空间：

```
图像特征 (196 × 768)
     │
     ▼ 线性投影 / MLP / Q-Former
     │
视觉 Token (196 × 4096)  ← 和文字 Token 一样维度
```

### 3. LLM 骨干

处理视觉 Token 和文字 Token，生成回答。

## 代表性模型

## 训练方法

### 预训练

```
数据: (图像, 文本) 对
- 图像描述数据
- 网络图文对
- 交错图文数据

任务: 下一个 Token 预测
输入: [图像特征][文本前缀]
目标: 预测后续文本
```

## 应用场景

| 场景 | 描述 | 示例 |
|------|------|------|
| **图像描述** | 自动生成图片描述 | 相册自动标注 |
| **视觉问答** | 回答关于图像的问题 | 产品咨询 |
| **OCR** | 识别图中文字 | 文档数字化 |
| **图表理解** | 理解图表数据 | 数据分析 |
| **内容审核** | 检测不当内容 | 社交媒体 |

## 总结

| 概念 | 通俗理解 |
|------|----------|
| 视觉语言模型 | 能看图说话的 AI |
| 视觉编码器 | 把图像变成向量 |
| 适配器 | 把视觉向量"翻译"成语言向量 |
| LLaVA | 经典开源 VLM |
| BLIP-2 | 用 Q-Former 高效连接视觉和语言 |
| GPT-4V | 最强闭源多模态模型 |
