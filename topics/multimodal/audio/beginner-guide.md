# 音频处理（Audio）科普版

> 面向零基础读者的音频处理入门指南

## 一句话概括

**音频处理让 AI "听懂"声音，实现语音识别、语音合成等功能。**

## 从一个问题开始

怎么让 AI 听懂人说的话？

**语音识别 (ASR):** 声音 → 文字
**语音合成 (TTS):** 文字 → 声音

## 核心概念

### 1. 音频 → 频谱图

声音是波形，但 AI 更容易处理"图像"：

```
原始音频 (波形)              频谱图 (类似图像)
┌─────────────────┐         ┌─────────────────┐
│    ∿∿∿∿∿∿∿     │         │ ▓▓░░▓▓░░▓▓░░   │
│  ∿∿      ∿∿∿∿  │   ──▶   │ ░░▓▓░░▓▓░░▓▓   │
│∿∿            ∿∿│         │ ▓▓▓▓░░░░▓▓▓▓   │
└─────────────────┘         └─────────────────┘
   时间域                     时频域

频谱图 = X 轴时间 + Y 轴频率 + 颜色强度
```

### 2. 梅尔频谱 (Mel Spectrogram)

模拟人耳对不同频率的敏感度：

```
人耳特点:
- 对低频敏感 (容易区分 100Hz 和 200Hz)
- 对高频不敏感 (难区分 10000Hz 和 10100Hz)

梅尔刻度: 把频率"扭曲"成更符合人耳感知的方式
```

## Whisper

OpenAI 开源的语音识别模型。

### 架构

```
┌─────────────────────────────────────────────────────────────────┐
│                     Whisper 架构                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   输入: 音频 (30秒)                                             │
│         │                                                       │
│         ▼                                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  Log-Mel Spectrogram                                     │   │
│   │  音频 → 频谱图 (80 × 3000)                               │   │
│   └─────────────────────────────────────────────────────────┘   │
│         │                                                       │
│         ▼                                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  Encoder (Transformer)                                   │   │
│   │  提取音频特征                                             │   │
│   └─────────────────────────────────────────────────────────┘   │
│         │                                                       │
│         ▼                                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  Decoder (Transformer)                                   │   │
│   │  自回归生成文本                                           │   │
│   └─────────────────────────────────────────────────────────┘   │
│         │                                                       │
│         ▼                                                       │
│   输出: 文字 "今天天气怎么样？"                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 特点

| 特点 | 说明 |
|------|------|
| **多语言** | 支持 99 种语言 |
| **多任务** | 识别 + 翻译 + 检测语言 |
| **鲁棒性** | 抗噪音、口音 |
| **开源** | 完全开源可用 |

### 使用示例

```python
import whisper

# 加载模型
model = whisper.load_model("base")

# 识别语音
result = model.transcribe("audio.mp3")
print(result["text"])  # 输出识别的文字
```

## 语音合成 (TTS)

### 基本流程

```
┌─────────────────────────────────────────────────────────────────┐
│                     TTS 流程                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   输入文本: "你好世界"                                          │
│         │                                                       │
│         ▼                                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  文本处理                                                 │   │
│   │  - 分词                                                   │   │
│   │  - 音素转换 (Grapheme-to-Phoneme)                        │   │
│   │  - 韵律预测                                               │   │
│   └─────────────────────────────────────────────────────────┘   │
│         │                                                       │
│         ▼                                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  声学模型                                                 │   │
│   │  文本特征 → 频谱图                                        │   │
│   └─────────────────────────────────────────────────────────┘   │
│         │                                                       │
│         ▼                                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  声码器 (Vocoder)                                         │   │
│   │  频谱图 → 音频波形                                        │   │
│   └─────────────────────────────────────────────────────────┘   │
│         │                                                       │
│         ▼                                                       │
│   输出: 🔊 音频                                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 常见 TTS 模型

| 模型 | 特点 |
|------|------|
| **Tacotron 2** | 经典端到端 TTS |
| **FastSpeech 2** | 非自回归，快速 |
| **VITS** | 端到端，高质量 |
| **Bark** | 基于 GPT 的 TTS |
| **GPT-4o-audio** | 多模态语音 |

### 语音克隆

用少量样本复制某个人的声音：

```
训练: 30秒某人的语音样本
     │
     ▼
┌─────────────────────────────────────────────────────────────┐
│                    语音克隆模型                               │
│                                                             │
│   提取说话人特征 (Speaker Embedding)                         │
│   +                                                          │
│   文本 → 合成 → 该说话人声音的音频                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
     │
     ▼
输出: 用那个人的声音说任何话
```

## 总结

| 概念 | 通俗理解 |
|------|----------|
| 频谱图 | 把声音"画"成图 |
| ASR | 听写：声音变文字 |
| TTS | 朗读：文字变声音 |
| Whisper | OpenAI 开源语音识别 |
| 语音克隆 | 复制某人的声音 |
| 端到端语音 | 直接语音对话，不经过文字 |
