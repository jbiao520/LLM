```mermaid
flowchart LR
    A[输入: 今天天气] --> B[模型]
```

## 采样策略对比

```mermaid
graph TB
    subgraph 贪婪 "Greedy Decoding"
        A1[选择概率最高的词] --> A2[确定性输出]
    end

    subgraph 温度 "Temperature Sampling"
        B1[调整概率分布] --> B2[按新分布采样]
    end

    subgraph TopK "Top-K Sampling"
        C1[保留 Top K 个词] --> C2[重新归一化]
        C2 --> C3[采样]
    end

    subgraph TopP "Top-P Sampling"
        D1[保留累计概率 ≥ P] --> D2[重新归一化]
        D2 --> D3[采样]
    end

    subgraph Beam "Beam Search"
        E1[维护 K 条路径] --> E2[每步扩展]
        E2 --> E3[保留最优 K 条]
    end
```

## 完整生成流程

```
┌─────────────────────────────────────────────────────────────────┐
│                      完整的文本生成流程                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  1. 输入处理                                              │  │
│   │  "今天天气怎么样？" → tokenizer → [101, 456, 789, 102]   │  │
│   └─────────────────────────────────────────────────────────┘  │
│                             │                                   │
│                             ▼                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  2. 模型前向传播                                          │  │
│   │  tokens → Transformer → logits                          │  │
│   │  logits: [2.1, 1.5, 0.8, ...] (vocab_size 维)           │  │
│   └─────────────────────────────────────────────────────────┘  │
│                             │                                   │
│                             ▼                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  3. 应用 Temperature                                      │  │
│   │  logits = logits / temperature                          │  │
│   └─────────────────────────────────────────────────────────┘  │
│                             │                                   │
│                             ▼                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  4. 应用 Top-K 和/或 Top-P                                │  │
│   │  过滤低概率 token                                         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                             │                                   │
│                             ▼                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  5. 应用重复惩罚                                          │  │
│   │  降低已出现 token 的概率                                  │  │
│   └─────────────────────────────────────────────────────────┘  │
│                             │                                   │
│                             ▼                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  6. Softmax                                              │  │
│   │  logits → 概率分布                                       │  │
│   └─────────────────────────────────────────────────────────┘  │
│                             │                                   │
│                             ▼                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  7. 采样                                                  │  │
│   │  multinomial(probabilities) → token_id                  │  │
│   └─────────────────────────────────────────────────────────┘  │
│                             │                                   │
│                             ▼                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  8. 检查停止条件                                          │  │
│   │  - 是否遇到 EOS token?                                   │  │
│   │  - 是否达到 max_length?                                  │  │
│   │  - 是否遇到 stop sequence?                               │  │
│   └─────────────────────────────────────────────────────────┘  │
│                             │                                   │
│                    ┌────────┴────────┐                         │
│                    ▼                 ▼                         │
│               [继续生成]         [停止输出]                     │
│                    │                 │                         │
│                    └────────┬────────┘                         │
│                             ▼                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  9. 解码输出                                              │  │
│   │  token_ids → tokenizer.decode → "今天天气不错"           │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
