# 稀疏注意力（Sparse Attention）科普版

> 面向零基础读者的稀疏注意力入门指南

## 一句话概括

**稀疏注意力让模型只关注"重要的词"，而不是所有词，从而处理更长的文本。**

## 从一个问题开始

标准注意力有一个致命问题：**计算量随长度平方增长！**

```
序列长度 1000:   1,000,000 次计算 ✓
序列长度 4000:   16,000,000 次计算 △
序列长度 16000:  256,000,000 次计算 ✗
序列长度 64000:  4,096,000,000 次计算 ✗✗✗
```

**怎么办？** 答案就是稀疏注意力！

## 类比：阅读理解

```
┌─────────────────────────────────────────────────────────────────┐
│                     阅读理解 vs 注意力                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   文章：                                                        │
│   "小明今年15岁，住在北京海淀区，喜欢打篮球和弹钢琴..."            │
│                                                                 │
│   标准注意力（读完每个字都关注所有字）：                           │
│   读"15岁"时，回顾所有之前的字                                   │
│   ┌─────────────────────────────────────────┐                   │
│   │ 小明今年15岁住在北京海淀区...            │                   │
│   │ ████████████████████████████████        │ 全部关注          │
│   └─────────────────────────────────────────┘                   │
│                                                                 │
│   稀疏注意力（只关注相关的）：                                    │
│   读"15岁"时，只回顾"小明"                                       │
│   ┌─────────────────────────────────────────┐                   │
│   │ 小明今年15岁住在北京海淀区...            │                   │
│   │ █   █    █                              │ 选择性关注        │
│   └─────────────────────────────────────────┘                   │
│                                                                 │
│   人类阅读就是这样：不会逐字回顾，而是抓住关键                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 核心概念

### 注意力矩阵

标准注意力的注意力矩阵是稠密的（所有位置都有值）：

```
标准注意力矩阵 (8×8):
┌────────────────────────────────────────┐
│ █ █ █ █ █ █ █ █ │  每个位置关注所有位置
│ █ █ █ █ █ █ █ █ │  计算量: O(n²)
│ █ █ █ █ █ █ █ █ │
│ █ █ █ █ █ █ █ █ │
│ █ █ █ █ █ █ █ █ │
│ █ █ █ █ █ █ █ █ │
│ █ █ █ █ █ █ █ █ │
│ █ █ █ █ █ █ █ █ │
└────────────────────────────────────────┘
```

### 稀疏注意力

稀疏注意力只保留重要的连接：

```
稀疏注意力矩阵 (8×8):
┌────────────────────────────────────────┐
│ █     █     █     █ │  大部分为 0
│   █     █     █     │  计算量: O(n)
│ █ █ █       █ █ █ │  或 O(n√n)
│       █ █ █       │
│ █     █     █     █ │
│   █     █     █     │
│ █ █ █       █ █ █ │
│       █ █ █       │
└────────────────────────────────────────┘
```

## 三种稀疏模式

### 1. 局部注意力（Local Attention）

只关注附近的词，像滑动窗口：

```
┌─────────────────────────────────────────────────────────────────┐
│                     局部注意力（窗口=3）                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   序列: [A, B, C, D, E, F, G, H]                                │
│                                                                 │
│   位置 C 的注意力:                                               │
│   ┌───────────────────────────────────────┐                     │
│   │  A   B   C   D   E   F   G   H       │                     │
│   │      ●   ●   ●                        │  只看 B, C, D      │
│   └───────────────────────────────────────┘                     │
│                                                                 │
│   注意力矩阵:                                                    │
│   ┌────────────────────────────────────────┐                    │
│   │ █ █                                    │                    │
│   │ █ █ █                                  │                    │
│   │   █ █ █                                │                    │
│   │     █ █ █                              │                    │
│   │       █ █ █                            │                    │
│   │         █ █ █                          │                    │
│   │           █ █ █                        │                    │
│   │             █ █                        │                    │
│   └────────────────────────────────────────┘                    │
│                                                                 │
│   计算量: O(n × w)，w 是窗口大小                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 全局注意力（Global Attention）

某些特殊位置（如 [CLS]）关注所有位置：

```
┌─────────────────────────────────────────────────────────────────┐
│                     全局注意力                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   序列: [CLS, A, B, C, D, E, F, SEP]                            │
│          ↑                        ↑                             │
│        全局位置                  全局位置                        │
│                                                                 │
│   注意力矩阵:                                                    │
│   ┌────────────────────────────────────────┐                    │
│   │ █ █ █ █ █ █ █ █ │  CLS 关注所有       │                    │
│   │ █ █ █                                  │                    │
│   │ █   █ █                                │                    │
│   │ █     █ █                              │                    │
│   │ █       █ █                            │                    │
│   │ █         █ █                          │                    │
│   │ █           █ █                        │                    │
│   │ █ █ █ █ █ █ █ █ │  SEP 关注所有       │                    │
│   └────────────────────────────────────────┘                    │
│                                                                 │
│   Longformer, BigBird 使用这种模式                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 随机注意力（Random Attention）

随机选择一些位置关注，增加多样性：

```
┌─────────────────────────────────────────────────────────────────┐
│                     随机注意力                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   位置 C 随机选择 2 个位置关注:                                   │
│   ┌───────────────────────────────────────┐                     │
│   │  A   B   C   D   E   F   G   H       │                     │
│   │  ●               ●                    │  随机选了 A, E     │
│   └───────────────────────────────────────┘                     │
│                                                                 │
│   作用: 增加长距离连接，打破局部限制                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 代表性模型

### Longformer

组合三种模式：
- 局部滑动窗口（默认 w=512）
- 全局注意力（特定位置）
- 扩张滑动窗口（跳着看）

**效果：** 可以处理 16K+ 长度

### BigBird

组合三种模式：
- 随机注意力
- 局部窗口注意力
- 全局注意力

**理论证明：** 这种组合近似完整注意力

### LongNet (Dilated Attention)

扩张注意力：间隔采样

```
普通: 每1个看1个  [A, B, C, D, E, F, G, H]
扩张: 每2个看1个  [A,   C,   E,   G]

距离近的：密集采样
距离远的：稀疏采样
```

## 稀疏注意力的效果

| 模型 | 序列长度 | 计算量 | vs 标准 |
|------|----------|--------|---------|
| 标准 | 4K | O(16M) | 1x |
| Longformer | 16K | O(16M) | 1x (4x长度) |
| BigBird | 16K | O(16M) | 1x (4x长度) |
| LongNet | 1M | O(n) | 可处理100万长度 |

**关键优势：** 线性复杂度，可以处理超长文本！

## 总结

| 概念 | 通俗理解 |
|------|----------|
| 稀疏注意力 | 只关注重要的词 |
| 局部注意力 | 滑动窗口，只看附近 |
| 全局注意力 | 特殊位置看全部 |
| 随机注意力 | 随机选几个位置 |
| 计算量 | 从 O(n²) 降到 O(n) |
