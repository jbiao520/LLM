# 分词器流程图解

## BPE 训练流程

```mermaid
flowchart TD
    A[原始语料] --> B[拆分为字符]
    B --> C[统计相邻对频率]
    C --> D{最高频对}
    D --> E[合并为新 token]
    E --> F{达到目标词表大小?}
    F -->|否| C
    F -->|是| G[输出词汇表]
```

## 编码流程

```mermaid
flowchart LR
    A[输入文本] --> B[拆分为字符]
    B --> C[查找可合并对]
    C --> D{找到可合并对?}
    D -->|是| E[合并]
    E --> C
    D -->|否| F[查找 token ID]
    F --> G[输出 token IDs]
```

## 解码流程

```mermaid
flowchart LR
    A[Token IDs] --> B[查找 token 字符串]
    B --> C[拼接 tokens]
    C --> D[处理特殊字符]
    D --> E[输出文本]
```

## BPE 合并示例

```
训练语料: "low lower newest widest"

初始状态 (字符级别):
┌─────────────────────────────────────┐
│ l o w (5次)                         │
│ l o w e r (2次)                     │
│ n e w e s t (6次)                   │
│ w i d e s t (3次)                   │
└─────────────────────────────────────┘

第1轮: 合并 "e" + "s" = "es" (9次)
┌─────────────────────────────────────┐
│ l o w (5次)                         │
│ l o w e r (2次)                     │
│ n e w es t (6次)                    │
│ w i d es t (3次)                    │
└─────────────────────────────────────┘

第2轮: 合并 "es" + "t" = "est" (9次)
┌─────────────────────────────────────┐
│ l o w (5次)                         │
│ l o w e r (2次)                     │
│ n e w est (6次)                     │
│ w i d est (3次)                     │
└─────────────────────────────────────┘

第3轮: 合并 "l" + "o" = "lo" (7次)
...
```

## Token 长度与成本

```
┌────────────────────────────────────────────────────────────────┐
│              文本长度 vs Token 数量 vs API 成本                 │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  英文 (1 token ≈ 4 chars)                                      │
│  "The quick brown fox jumps over the lazy dog"                 │
│  Length: 44 chars → ~10 tokens                                 │
│                                                                │
│  中文 (1 token ≈ 1-2 chars)                                    │
│  "那只敏捷的棕色狐狸跳过了懒狗"                                   │
│  Length: 15 chars → ~12-15 tokens                              │
│                                                                │
│  代码 (tokenization 较差)                                       │
│  "def fibonacci(n): return n if n < 2 else ..."               │
│  可能有更多 tokens（取决于空格和符号）                            │
│                                                                │
│  ─────────────────────────────────────────────────────────────│
│                                                                │
│  成本计算 (假设 $0.01/1K tokens):                               │
│  1000 英文单词 ≈ 1300 tokens ≈ $0.013                          │
│  1000 中文字   ≈ 1500 tokens ≈ $0.015                          │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```
